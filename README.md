# Project RE: Hardware-Anchored AI Governance Protocol

> Make Intelligence Accountable. Artificial or Otherwise.

## What is this?

Demo app for the Devpost x Google DeepMind Gemini 3 Hackathon 2026.
Project RE is a hardware-anchored governance protocol 
for autonomous AI agents.

## Quick Start (Local)
```bash
npm install
npm run dev
```

Requires: Node.js 18+, Gemini API key in `.env`

React, TypeScript, Tailwind CSS, Gemini API 
(gemini-3-flash-preview / gemma-3-27b-it)
